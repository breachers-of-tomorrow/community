# Contribution Paths

There are many ways to contribute to Breachers of Tomorrow. You don't need to be a developer — some of the most valuable contributions come from research, writing, and community engagement.

## 🔍 ARG Research

**Best for:** Anyone following the Marathon ARG

- Track changes on [cryoarchive.systems](https://cryoarchive.systems)
- Document discoveries in the [Community Doc](https://docs.google.com/document/d/1mtUtDPvbh6ahiynYFVS7Z4O79Nw6y5PEOjweCpzWV_A/edit?tab=t.0)
- Share findings in Discord `#general` or the Thinking Pods voice channels
- Help organize research into structured wiki pages (coming soon)

## 💻 Code & Tools

**Best for:** Developers, designers, data enthusiasts

- **[breacher-net](https://github.com/breachers-of-tomorrow/breacher-net)** — Our Next.js dashboard. Check [open issues](https://github.com/breachers-of-tomorrow/breacher-net/issues) or the [CONTRIBUTING guide](https://github.com/breachers-of-tomorrow/breacher-net/blob/main/CONTRIBUTING.md)
- Build data visualizations or analysis tools
- Help with the poller service that tracks ARG changes
- Contribute to Code Corner discussions in Discord

**Tech stack:** Next.js 16, React 19, TypeScript 5, Tailwind CSS 4, PostgreSQL, Python (poller)

## 📝 Writing & Documentation

**Best for:** Writers, editors, organizers

- Help migrate research from the Google Doc to structured wiki pages
- Write guides for new members
- Document ARG theories with evidence
- Improve existing documentation

## 🎨 Design & Branding

**Best for:** Designers, artists, creative types

- Help develop the community's visual identity
- Create social media graphics and event banners
- Design Discord server assets (icon, banner, role colors)
- Contribute to the [Brand Guide](../branding/BRAND-GUIDE.md)

## 🎙️ Community Building

**Best for:** Social people, event organizers, moderators

- Help welcome new members in Discord
- Organize community events (raid nights, ARG solving sessions)
- Moderate discussions and help maintain community standards
- Share community content on social media

## 📊 Data & Analysis

**Best for:** Data enthusiasts, spreadsheet warriors

- Track sector data, kill counts, and stabilization patterns
- Build and maintain the [ARG Spreadsheet](https://docs.google.com/spreadsheets/)
- Create charts and visualizations of ARG trends
- Help validate theories with data evidence

## 🚀 Getting Started

1. **Pick what interests you** from the paths above
2. **Join the relevant Discord channels** — `#general` for research, `Code Corner` for dev work
3. **Start small** — Fix a typo, add a data point, answer a question
4. **Ask for help** — The Keepers and community are here to support you

No contribution is too small. Every data point tracked, every question answered, and every discovery shared makes the community stronger.
