# How We Work

> **Last updated:** 2026-03-12

## Decision Making

Community decisions follow this pattern:

1. **Discussion** — Ideas start in [GitHub Discussions](https://github.com/breachers-of-tomorrow/community/discussions/categories/ideas) or Discord
2. **Deliberation** — Keepers discuss in `#keepers-of-the-manifest` and/or `community-ops`
3. **Decision** — Keepers reach consensus and document the decision
4. **Communication** — Decision is announced in `#announcements` and relevant docs are updated

## Tools We Use

| Tool | Purpose | Who Uses It |
|------|---------|-------------|
| Discord | Real-time communication, voice, events | Everyone |
| GitHub (community repo) | Persistent discussions, docs, processes | Everyone |
| GitHub (community-ops repo) | Leadership coordination | Keepers only |
| GitHub (breacher-net repo) | Website development | Contributors |
| breacher.net | Live ARG dashboard | Everyone (view) |
| Google Doc | Active ARG investigation notes | Editors |
| Wiki (future) | Curated knowledge base | Everyone (read), Keepers (publish) |

## Communication Channels

- **Quick questions / casual chat** → Discord `#general`
- **Persistent community discussions** → [GitHub Discussions](https://github.com/breachers-of-tomorrow/community/discussions)
- **Bug reports / feature requests** → [breacher-net Issues](https://github.com/breachers-of-tomorrow/breacher-net/issues)
- **Community improvement ideas** → [Ideas Discussion](https://github.com/breachers-of-tomorrow/community/discussions/categories/ideas)
- **Leadership coordination** → `#keepers-of-the-manifest` on Discord + `community-ops` repo
