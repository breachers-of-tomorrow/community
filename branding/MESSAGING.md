> **Breachers of Tomorrow** is a 1,000+ member community tracking the Marathon ARG. We build tools, solve puzzles, and prepare for what comes next.

- "Community tracker and hub for the Marathon ARG"

## Voice & Tone

### How We Sound

- **In-universe but not cringe** — We use ARG terminology ("breach protocol", "sectors", "stabilization") naturally, not forced
- **Collaborative** — "We discovered" not "I found"
- **Approachable** — Experts welcome, but never gatekeeping
- **Enthusiastic but grounded** — Excited about discoveries, skeptical about unconfirmed theories

### Channel-Specific Tone

| Context | Tone |
|---------|------|
| Discord `#announcements` | Clear, concise, slightly formal |
| Discord `#general` | Casual, friendly, inclusive |
| GitHub Issues/PRs | Technical, constructive |
| GitHub Discussions | Thoughtful, longer-form |
| Wiki content | Neutral, factual, well-sourced |
| Social media | Energetic, visual, shareable |
