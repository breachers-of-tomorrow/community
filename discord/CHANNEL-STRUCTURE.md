# Discord Channel Structure

> **Last updated:** 2026-03-17
> **Server:** Breachers of Tomorrow

## Server Overview

The Discord server is our real-time community hub with 1,500+ members. It's organized into categories for ARG research, general conversation, creative work, development, and voice hangouts.

## Channel Moderation Notes

| Category | Moderation Level | Notes |
|----------|-----------------|-------|
| Important | Low traffic, Keeper-managed | Mostly read-only or Keeper-posted |
| ARG Posting | Medium — spoiler discipline | Enforce spoiler tags, source claims |
| General Posting | Highest volume — primary monitoring | Most rule violations happen here |
| Art Corner | Low — community-positive | Light touch, flag NSFW if any |
| Code Corner | Low — self-regulating | Technical community, rarely needs mod action |
| Thinking Pods | Medium — voice moderation | Watch for disruptive audio, soundboards |
| Runner Crews | Low | Self-organizing raid groups |
