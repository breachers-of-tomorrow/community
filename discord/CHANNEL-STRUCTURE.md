# Discord Channel Structure

> **Last updated:** 2026-03-12
> **Server:** Breachers of Tomorrow

## Server Overview

The Discord server is our real-time community hub with 1,000+ members. It's organized into sections for different activities.

## Channel Layout

### 📌 Top-Level

| Feature | Purpose |
|---------|---------|
| Server Guide | New member orientation |
| Events | Scheduled community events |
| Browse Channels | Channel discovery |
| Members | Member list |

### ℹ️ Important

| Channel | Purpose |
|---------|---------|
| `#rules` | Community guidelines and expectations |
| `#announcements` | Official community updates (Keepers only) |
| `#roles` | Self-assignable role selection |
| `#community-arg-google-doc` | Link to collaborative ARG research document |
| `#list-of-editors` | People with edit access to shared docs |
| `#the-best-posts` | Curated standout community contributions |

### 💬 Text Channels

| Channel | Purpose |
|---------|---------|
| `#general` | Main community discussion |
| `#lfg-breach-protocol` | Looking for group / find teammates |
| `#offtopic` | Non-ARG conversation |
| `#photos-and-discussion` | Image sharing and visual discussion |

### 🔐 Keepers of The Manifest (Private)

| Channel | Purpose |
|---------|---------|
| `#keepers-of-the-manifest` | Leadership coordination and discussion |
| `#reports` | Moderation reports and action items |
| Keepers Voice Chat | Private voice channel for leadership calls |

### 🎙️ Voice Channels

| Channel | Purpose |
|---------|---------|
| Thinking Pods (20+ slots) | General voice hangout and ARG discussion |
| Runner Groups (VC) | Gameplay voice channels |

### 📡 Other

| Channel | Purpose |
|---------|---------|
| Protocol Updates | ARG status changes and news |
| Code Corner / Code Chats | Developer discussion and live coding |

## Future Additions

- **`#github`** — Webhook-fed notifications for new issues, PRs, and discussions
- **`#wiki`** — Wiki updates and editorial coordination (when wiki launches)
- Consider renaming `#community-arg-google-doc` to `#community-resources` when wiki goes live
